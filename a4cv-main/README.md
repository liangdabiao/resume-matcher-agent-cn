# A4 可视化简历生成器

一个本地优先的静态网页工具，用来把 Markdown、TXT 或 Word 简历内容快速整理成可视化 A4 简历。

## 功能概览

- 支持粘贴文本、导入 Markdown / TXT / DOCX 内容
- 内置多组风格预设和布局版式
- 支持头像上传、裁切比例、大小和位置调整
- 支持模块拖拽排序、左右栏宽度调整
- 支持字体大小、行距、边距、技能标签、照片形态等细节设置
- 支持将当前细节保存为对应风格的默认版式
- 支持导出 PNG、PDF、Markdown 和 Word 内容文档

## 本地使用

直接用浏览器打开 `index.html` 即可使用。项目依赖已放在 `vendor/` 目录，不需要联网安装。
